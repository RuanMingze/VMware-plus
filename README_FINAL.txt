VMware++ 修改器 v2.0
====================

简介
----
VMware++ 是一个用于修改 VMware 虚拟机配置文件(.vmx)的图形化工具，具有现代化的用户界面和完善的错误处理机制。

主要功能
--------
1. 为 macOS 虚拟机启用 SMC 仿真
   - 在 .vmx 文件中添加: smc.version = "0"
   - 改善 macOS 虚拟机的硬件兼容性

2. 为 macOS 的 AMD CPU 用户添加兼容层
   - 添加一系列 CPUID 和硬件模拟设置
   - 提升 AMD 处理器上 macOS 虚拟机的兼容性
   - 可与 SMC 仿真共存

3. 禁用自动挂起功能
   - 在 .vmx 文件中添加: suspend.disabled = "TRUE"
   - 防止虚拟机在后台运行时自动挂起

程序特性
--------
- 现代化图形界面：使用彩色主题，提供美观的视觉效果
- 智能文件处理：自动查找并处理指定目录下的所有 .vmx 文件
- 安全操作机制：处理文件时先重命名为 .txt 文件，处理完成后再恢复
- 智能检测功能：自动检测是否已存在配置项，避免重复添加
- 反向操作支持：取消勾选选项时会自动删除已添加的配置项
- 完善错误处理：提供详细的错误信息和日志记录
- 多种错误处理：包括文件不存在、访问权限不足、文件被占用等情况

使用方法
--------
1. 运行 VMware++.exe 程序
2. 点击"浏览..."按钮选择包含虚拟机文件的目录
3. 勾选需要应用的修改选项:
   - 启用 SMC 仿真: 适用于 macOS 虚拟机
   - 启用 AMD 兼容层: 适用于使用 AMD 处理器的 macOS 虚拟机
   - 禁用自动挂起: 适用于所有虚拟机
4. 点击"应用修改"按钮
5. 等待程序完成处理，查看操作日志确认修改结果

注意事项
--------
- 程序会自动查找选定目录及其子目录中的所有 .vmx 文件
- 程序会先将 .vmx 文件临时重命名为 .txt 文件，处理完成后再恢复为 .vmx 文件
- 如果配置项已存在，则不会重复添加
- 建议在修改前备份重要的虚拟机配置文件
- 修改后可能需要重新启动虚拟机才能生效

技术说明
--------
- 本工具使用 Python 3.x 编写，通过 Tkinter 提供图形界面
- 使用 PyInstaller 打包为独立的 Windows 可执行文件
- 不会修改除 .vmx 文件以外的任何文件
